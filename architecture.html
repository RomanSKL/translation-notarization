Request Flow

① Upload

Browser → Cloudflare → Nginx → Next.js. File accepted, job added to Redis queue. jobId returned immediately.

② Processing

Worker picks job from Redis → calls Claude API for translation → builds PDF → saves to MongoDB.

③ Download

Browser polls job status every 2s. When complete → download button appears → PDF served from MongoDB.
